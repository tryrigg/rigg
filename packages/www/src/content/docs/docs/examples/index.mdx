## Review & auto-fix loop

Reviews uncommitted changes and auto-fixes in a loop.

```yaml
id: review-uncommitted
steps:
  - id: remediation
    type: loop
    max: 5
    until: ${{ len(steps.review.result.findings) == 0 }}
    steps:
      - id: review
        type: codex
        with:
          action: review
          review:
            target:
              type: uncommitted

      - id: fix
        if: ${{ len(steps.review.result.findings) > 0 }}
        type: codex
        with:
          action: run
          prompt: |
            Address the accepted findings from this review.
            Review:
            ${{ toJSON(steps.review.result) }}
```

## Plan, review, and refine

Drafts a plan, critiques it, refines it, then saves.

```yaml
id: plan
inputs:
  requirements:
    type: string
  output_path:
    type: string
steps:
  - id: draft
    type: codex
    with:
      action: run
      prompt: |
        Draft an implementation plan:
        ${{ inputs.requirements }}

  - id: critique
    type: codex
    with:
      action: run
      prompt: |
        Review for gaps:
        ${{ steps.draft.result }}

  - id: refine
    type: codex
    with:
      action: run
      prompt: |
        Improve this plan using the critique.
        Draft:
        ${{ steps.draft.result }}
        Critique:
        ${{ steps.critique.result }}

  - id: save
    type: write_file
    with:
      path: ${{ inputs.output_path }}
      content: ${{ steps.refine.result }}
```

## Parallel checks

Runs multiple shell checks concurrently, then summarizes them with Codex.

```yaml
id: parallel-checks
steps:
  - id: all
    type: parallel
    branches:
      - id: unit
        steps:
          - id: run_unit
            type: shell
            with:
              command: npm test
              result: text
      - id: lint
        steps:
          - id: run_lint
            type: shell
            with:
              command: npm run lint
              result: text
    exports:
      unit: ${{ steps.run_unit.result }}
      lint: ${{ steps.run_lint.result }}

  - id: report
    type: codex
    with:
      action: run
      prompt: |
        Summarize:
        Unit: ${{ steps.all.result.unit }}
        Lint: ${{ steps.all.result.lint }}
```

## Commit review

```yaml
id: review-commit
inputs:
  commit_sha:
    type: string
steps:
  - id: review
    type: codex
    with:
      action: review
      review:
        target:
          type: commit
          sha: ${{ inputs.commit_sha }}
```

## Patterns

### Review-driven loop

Use review findings as the loop condition:

```yaml
until: ${{ len(steps.review.result.findings) == 0 }}
```

### Branch passthrough

```yaml
- else:
  steps: []
  exports:
    value: ${{ steps.earlier.result }}
```
